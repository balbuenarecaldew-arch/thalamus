Descargá un backup completo de todos los datos. Si algo se rompe, restaurá desde el archivo.
Backup automático: cada vez que ingresás se guarda una copia en Firebase. Se mantienen los últimos 5 backups.
🔒 Seguridad de Firebase
Tu Firestore puede estar en modo abierto. Para proteger tus datos, andá a la consola de Firebase → Firestore → Reglas, y pegá estas reglas:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; // Para más seguridad, habilitá Firebase Auth // y cambiá la línea de arriba por: // allow read, write: if request.auth != null; } } }
💡 Estas reglas permiten lectura/escritura. Para máxima seguridad, habilitá Firebase Authentication y cambiá las reglas como indica el comentario.
